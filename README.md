# bside-02

## 🖐 Requirements

**Node:**

- NodeJS >= 14
- NPM >= 7

## Getting Started

### ⏳ Installation

```shell
$ npm install --global npm
```

```shell
$ npm ci
```

If you need to cleaning, Remove npm modules(`node_modules`) directory from all packages.

```shell
$ npm run clean
```

### 💻 Development, Build

Run development server

```shell
$ npm run dev
```

Build packages

```shell
$ npm run build
```
